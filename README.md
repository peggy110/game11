# 小津琳迷宮遊戲

這是一個用 HTML5 + JavaScript 製作的迷宮遊戲：
- 每一關主角小津琳都要走到角落吃掉圓形物品。
- 每通過一關，迷宮會自動變大、變難。
- 迷宮保證有解，主角一定能通關。

## 遊玩方式
1. 下載或 clone 此專案。
2. 直接用瀏覽器打開 `index.html`。
3. 用鍵盤方向鍵移動小津琳。

## 推薦功能
- 每關自動升級難度。
- 迷宮自動生成且保證有解。
